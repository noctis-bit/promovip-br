# 🚀 PromovipBR

PromovipBR é uma plataforma web desenvolvida para agregação e busca de ofertas de produtos, permitindo que usuários encontrem promoções organizadas de forma prática e eficiente.

O projeto foi criado com o objetivo de aplicar conhecimentos em desenvolvimento web, arquitetura de aplicações e automação de dados, unindo tecnologia e visão de negócio.

---

## 📖 Sobre o Projeto

O PromovipBR funciona como um buscador de ofertas, centralizando links de produtos e organizando informações para facilitar a experiência do usuário na busca por melhores preços.

Além do site, o ecossistema do projeto inclui uma automação integrada ao WhatsApp que utiliza IA para gerar descrições promocionais de forma automática.

---

## 🛠️ Tecnologias Utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- APIs para manipulação de dados

### Automação
- Node.js
- Integração com planilhas (Excel) para leitura de produtos
- Integração com IA (Google Gemini) para geração automática de textos promocionais

### Deploy
- Vercel

---

## 🏗️ Arquitetura do Projeto


O projeto segue uma separação clara entre frontend e backend, facilitando manutenção e escalabilidade.

---

## ⚙️ Funcionalidades

- 🔎 Busca e organização de ofertas
- 📦 Estruturação e categorização de produtos
- ⚡ Integração entre frontend e backend
- 🤖 Geração automática de anúncios com IA
- 📲 Automação de envio para grupos via WhatsApp

---

## 🤖 Automação com IA

O sistema de automação:

1. Lê dados de produtos a partir de planilhas (Excel)
2. Processa informações via Node.js
3. Utiliza IA (Gemini) para gerar descrições persuasivas
4. Envia automaticamente os anúncios formatados para grupos

Isso reduz trabalho manual e otimiza o fluxo de divulgação de promoções.
